`policyengine zenodo-mirror <country>` deposits a certified release's certification record (bundle manifest, bundle TRACE TRO, data release manifest) on Zenodo for preservation, returning `PreservationMirror` entries with real DOIs; dataset bytes are only deposited on explicit request and never from private source repos. A release's record-level preservation DOI (`DataReleaseManifest.preservation_dois`) is surfaced on the bundle TRO performance node as `pe:preservationDoi`, so a reader holding only the TRO can find and cite the preserved copy.
